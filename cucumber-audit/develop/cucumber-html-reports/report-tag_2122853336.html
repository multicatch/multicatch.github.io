Scenario Known Software Vulnerabilities Disclosure (Headers)
The "Server" and "X-Powered-By" headers provide information about technology that is used on the server side. They usually contain the software version (eg. "Apache/2.2.15 (CentOS) ...") and can be used to find known vulnerabilities of that software. Disabling them makes it more difficult to exploit the server software.
